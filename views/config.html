Config

Not quite static v2 uses a json file for its config
The config is a object of string keys and RouteObject
The 404 route will be used if a route can't be found

RouteObject

html: a path to a html file or undefined, if layout is set it will be applied to #body if not then #app
layout: a path to a html file or undefined
script: a path to a script file or undefined
module: a path to a module file or undefined, if html is not set then the render method will be used to render the page
isRegex: bool or undefined, if true then the path will be treated as regex
